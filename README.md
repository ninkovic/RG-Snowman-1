***

**Komande**

| Komanda       |   Opis             |
|   :---:       |   :---:            |
|   g           |  pokretanje igre   |
|   p           |  pauziranje        |

**Opis**
* Snesko skuplja grudve i uvecava se nakon 5 skupljenih.
* Snesko ima 3 zivota.
* Vrele kugle donose negativne bodove i smanjuju zivote.
|   kraj igre   |                                       |
|               | ako izadje sa ploce po kojoj se krece |
|               | ako izgubi sve zivota                 |
|               | ako poeni budu manji od 0             |
