# RG080-snowman

**Komande**

| Komanda       |   Opis             |
|   :---:       |   :---:            |
|   g           |  pokretanje igre   |
|   p           |  pauziranje        |
|   a           |  kretanje u levo   |
|   d           |  kretanje u desno  |
|   w           |  kretanje gore     |
|   s           |  kretanje dole     |
|   Esc         |  izlazak iz igrice |

**Opis**
* Snesko skuplja grudve i uvecava se nakon 5 skupljenih.
* Snesko ima 3 zivota.
* Vrele kugle donose negativne bodove i smanjuju zivote.
* Igrica se zavrsava kada: 
                           1. izadje sa ploce po kojoj se krece
                           2. izgubi sva tri zivota
                           3. bodovi budu manji od 0
